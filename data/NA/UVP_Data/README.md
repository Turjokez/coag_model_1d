# Raw UVP Data

Source: EXPORTSNA UVP data.

Main author / PI: Andrew McDonnell / Lee Karp-Boss.

Data in this folder:
- Raw UVP per-cast files.
- These are useful if we need to check the processed UVP products.

Model use:
- Not the first choice for model input.
- Use the processed UVP `.sb` files first because they already give particle size spectra.
- Keep raw UVP as a backup and quality check.

First processing step:
- Do not parse this first.
- Start with `data/NA/uvp/raw`.
