# Estapa Optical Sediment Trap Flux

Source: SeaBASS EXPORTSNA, University of Maine Estapa archive.

Main author / PI: Margaret Estapa.

Data in this folder:
- Optical sediment trap flux files.
- Includes STT and NBST flux products.

Model use:
- Use this for time-dependent particle flux comparison.
- Good for checking whether model export changes with time in the right way.
- Useful after the model is forced with real surface PSD.

First processing step:
- Parse flux time series by depth.
- Compare with model bottom or mid-depth flux at matching times.
