5. Configuration Parameters

All model options are controlled through the SimulationConfig class:

cfg = SimulationConfig();
cfg.n_sections     = 30;
cfg.alpha          = 0.10;
cfg.sinking_law    = 'kriest_8';
cfg.enable_disagg  = true;
sim = ColumnSimulation(cfg, col_grid, prof);
