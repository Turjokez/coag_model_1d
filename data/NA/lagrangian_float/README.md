# Lagrangian Float CTD and Oxygen

Source: SeaBASS, UWASH D'Asaro archive.

Main author / PI: Eric D'Asaro.

Data in this folder:
- Float CTD, oxygen, and SUNA files.
- These files have continuous float observations.

Model use:
- Use as time-varying physical forcing or a check on CTD profiles.
- Oxygen can help check microbial respiration context.
- Chlorophyll or related float data can help identify mixed layer and surface bloom timing.

First processing step:
- Verify that these files are the correct region for this run.
- Some filenames include `EXPORTSNP`, so check before using them in the North Atlantic model.
