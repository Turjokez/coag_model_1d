# 234Th and Size-Fractionated Flux

Source: SeaBASS EXPORTSNA, WHOI Buesseler archive.

Main author / PI: Ken Buesseler.

Data in this folder:
- 234Th survey data.
- In situ pump / size-fractionated particle data.

Model use:
- Use as an independent check on transfer efficiency.
- Useful for checking total POC export and size-specific export.
- Good benchmark for whether model export is too high or too low.

First processing step:
- Extract flux or particle concentration by depth and size fraction.
- Compare model transfer efficiency with 234Th-based transfer efficiency.
