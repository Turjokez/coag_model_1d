# Particle-Associated Respiration

Source: SeaBASS EXPORTSNA, UCSB Santoro archive.

Main author / PI: Alyson Santoro.

Data in this folder:
- Particle-associated respiration data.
- 16S DNA and ammonium files are also present.

Model use:
- This is the most direct check for microbial loss on sinking particles.
- Use it to constrain the particle-attached part of `microbe_r0`.
- It may also help decide if microbial loss should be size-dependent.

First processing step:
- Parse respiration by depth and sample type.
- Compare with model particle loss from the microbial term.
